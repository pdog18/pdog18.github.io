智能监测Lite隐私政策

一、用户隐私协议

“智能监测Lite”版权所属杭州训机智能科技有限公司（简称“我们”）。
“智能监测Lite”是一款为投资人提供资产信息查阅、以及处置进度跟进的工具型App。 依据法律的规定，我们将在特定情形下收集、使用、保存和披露您的个人信息。以下条款描述了我们如何收集、使用、保存和披露您的个人信息。
“智能监测Lite”App所提供的相关信息仅供参考，并不保证这些信息适合于您。且此类信息无意替代或暗示任何专业的意见。

1、个人信息的收集

1.1 个人信息

我们使用、保存、使用和披露个人信息是为了帮助您创建账户、识别用户、回应反馈和邮件等行为。我们会向您提供关于资产信息方面服务、发送资产基础数据、数据分析等信息以及其他新闻材料，以为您提供更好的服务。

1.2 为实现资产的保密性与安全性，如果您想使用“智能监测Lite”App，您必须注册为“智能监测Lite”App用户并且于注册页面上提供相关的个人信息。您必须承诺和保证：
1.2.1 为实现“智能监测Lite”的服务目的，我们会通过技术手段收集您的个人信息，通过技术手段所收集的个人信息仅限于您的姓名、电话号码、身份证号等使用”智能监测Lite”App所必需的信息。
1.2.2 “智能监测Lite”App采取多平台异步开发方式，支持Android以及iOS等多类操作系统的使用。当您使用iOS以及Android系统时，您需要输入手机号码进行绑定使用。在您通过各种手机操作系统使用“智能监测Lite”App的过程中，“智能监测Lite”App所收集的信息仅限于您同意使用“智能监测Lite”App基本服务所必需的信息。

1.3 您提供信息的其他方式

我们将采用其他方式（例如：电子邮件、用户支持、反馈以及调查表）来收集您的个人信息。如果您将个人信息填写在其他开放的平台上，其他用户可能会获知您的信息。

1.4 您的其他信息
为了便于您使用本网站和”智能监测Lite”App，我们的服务器将会收集关于您的移动通讯设备的信息，例如型号、设备ID、互联网协议地址、您访问的时间、路由数据包、信息发送人和接收数据（但不是信息本身），以及其他类似的特征记录。我们将使用Cookies和类似统一资源定位器（URL）的导航数据来收集您访问的时间和日期、搜索和查看的信息等。

2、个人信息的使用

2.1 您的个人信息将只在本隐私政策及提前通知您的更新的政策所载明的范围内使用[不会用于其它目的]。
2.2 匿名汇总统计数据不是”智能监测Lite”App所定义的个人用户信息，我们将为多种目的，包括但不限于分析和使用模式的报告等，来保存和使用此类信息。

3、个人信息的披露

3.1 除非本隐私政策载明的有限披露，我们会妥善保存您的个人信息并不会泄露客户名单。
3.2 您于此授权，以下情形下，我们将会向第三方披露您的个人信息而无需通过您的同意：
3.2.1 如果我们披露这些信息是用来识别、联系或向可能对我们的权利或财产、网站用户或者其他任何人（包括其他任何人的权利和财产）造成损害之行为提起诉讼之必要；
3.2.2 法律调查之必要；
3.2.3 此类披露是法律规定或应向我们送达的传票或其他执法部门发出的令状之必要。

4、安全

4.1 对我们来说，个人信息的安全非常重要。我们将尽合理的商业努力保障您的个人信息安全，并实施合理的商业程序限制非授权访问、使用或者披露您的个人信息。尽管采取上述措施，但是您应当知晓本网站或”智能监测Lite”App不能完全避免与个人信息相关的安全危险。

5、您应当协助我们以保护您的个人信息安全。例如，不要泄露您的个人密码。

6、第三方网站

6.1 当您点击本网站链接的其他网站时，您已经离开我们的网站并访问其他网站，其他网站可能会收集您的个人信息或匿名数据，因此，我们无法控制、检查或者回应此类网站和网站内容。本网站/软件的互联网接入服务商、网络技术服务商、广告商和相关的链接网站也可能收集您的个人信息。本隐私权声明并不包含他们所做的信息收集。

Cookies是当您浏览网站时，网站发送给您的电脑和手机的少量信息。我们将在网站上使用会话Cookies（当您关闭浏览器的时候随之过期）和持久Cookies（除非您删除此类Cookies，否则将一直保存在您的电脑上）来向您提供更加个性化的交互服务。

8.1 本隐私政策容许调整，如果我们对使用您的个人信息方法做出任何实质性调整，我们将通过您提供的电子邮件地址，或通过网站的公告通知您。
